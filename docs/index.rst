Python-OC-Lettings Documentation
==================================

Bienvenue dans la documentation technique du projet **Python-OC-Lettings-FR**.

Ce projet est une application Django refactorisée en une architecture modulaire et moderne.
Le projet est structuré en trois parties principales :

- **oc_lettings_site** : La configuration globale et le point d'entrée de l’application.
- **lettings** : Gère les locations et les adresses.
- **profiles** : Gère les profils des utilisateurs.

La documentation couvre l'installation, l'utilisation, et la description des modules.
